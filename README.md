# Sales Performance Dashboard

## Project Overview
This Power BI dashboard analyzes sales performance across different regions, categories, customer segments, and years.

## Features
- Total Sales KPI
- Regional Sales Performance
- Sales by Category & Region
- Customer Segment Analysis
- Yearly Sales Trend
- Interactive Region Slicer

## Tools Used
- Power BI
- Microsoft Excel
